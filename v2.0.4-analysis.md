### 3.6 EscPosParser.cs (188줄)
**역할**: ESC/POS 바이너리 파싱 (B안: 텍스트 추출)

#### 주요 메서드
**Parse()** (라인 15-109)
- 입력: byte[] (ESC/POS 바이너리)
- 출력: string (추출된 텍스트)

#### 파싱 로직
1. **제어 시퀀스 필터링**
   - ESC (0x1B): SkipEscSequence() 호출 (커터, 정렬, 줄넘김 등)
   - GS (0x1D): SkipGsSequence() 호출 (바코드, 반전, 문자 크기 등)
   - DLE (0x10): 2바이트 스킵
   - FS (0x1C): 2바이트 스킵

2. **텍스트 추출**
   - 줄바꿈 (0x0A): 유지
   - 캐리지 리턴 (0x0D): 유지
   - ASCII (0x20~0x7E): 추출
   - 한글 (0x80이상): 2바이트 쌍으로 추출

3. **인코딩**
   - EUC-KR(CP949) 디코딩 (한글)
   - 실패 시 UTF-8 폴백
   - 실패 시 ASCII 폴백

#### 설계 평가
- ESC 시퀀스 17가지 처리 (완전함)
- GS 시퀀스 가변 길이 바코드 처리 (정교함)
- 한글 자동 감지 (EUC-KR 우선)
- 기획서 "B안" 정확히 구현
